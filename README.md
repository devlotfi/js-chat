## Requirements

- Node JS
- pnpm

## Install

```bash
$ pnpm i
```

## Development

First have to configure the ENV variables first on the server and client and run the database migrations

```bash
# Run in server folder
$ pnpm run start:dev

# Run in web folder
$ pnpm run dev
```
